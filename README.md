# JSONP Injection Lab

این ریپازیتوری یک **آزمایشگاه JSONP Injection** ساده و آموزشی است که برای تمرین حملات client-side طراحی شده است. هدف این lab آموزش نحوه سوءاستفاده از **callback غیرمجاز در JSONP** و مشاهده اثر آن در مرورگر است.

---

## 📁 محتویات پروژه
- `server.js` : سرور Node.js که API آسیب‌پذیر JSONP و یک frontend ساده را فراهم می‌کند.
- `test.html` : فایل آماده PoC که نشان می‌دهد چگونه داده‌های قربانی با JSONP قابل استخراج هستند.
- سایر فایل‌ها : وابستگی‌ها و فایل‌های جانبی مورد نیاز.

---

## 🚀 راه‌اندازی Lab

1. ابتدا Node.js و npm را نصب کنید.
2. ترمینال را در مسیر پروژه باز کنید:
   ```bash
   cd path/to/jsonp-lab
````

3. وابستگی‌ها را نصب کنید:

   ```bash
   npm install express
   ```
4. سرور را اجرا کنید:

   ```bash
   node server.js
   ```
5. مرورگر خود را باز کرده و به آدرس زیر بروید:

   ```
   http://localhost:3000/
   ```

> شما یک صفحه ساده با اطلاعات پروفایل خواهید دید که از طریق JSONP لود می‌شود.
